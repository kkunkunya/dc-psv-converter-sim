### 4.2 直流侧参数（参数计算说明）

**电容选型依据**：
- 每台整流器额定电流：2048×10³/1500 ≈ 1365 A
- 纹波电流按 40% 估算：I_ripple ≈ 550 A
- 允许电压纹波：1%×750 V = 7.5 V
- C ≥ I_ripple / (2π × f_sw × ΔV) = 550/(2π×5000×7.5) ≈ 2330 μF → 取标准值 **2200 μF**

**限流电感依据**：
- 目标：母线短路时电流上升斜率 ≤ 1500 kA/s（给保护约 2ms 响应窗口）
- L_rect = V_dc / (dI/dt) = 1500/1500000 = **1 mH**（每个变流器 DC 侧双侧对称）

**母线 RL 依据**：
- L_bus：短路仿真观测用，提供约 750 A/ms 上升斜率 → **2 mH**
- R_bus：阻尼系数 ζ ≈ 0.5~0.7，R_bus = 2ζ√(L/C_eq) ≈ **0.05 Ω**
- 注：R_bus 不能过大，否则稳态压降过大（1000A × 0.05Ω = 50V，在 1500V 系统中为 3.3%）

| 参数 | 初始推荐值 | 调整范围 | 在 Simulink 中的位置 |
|------|-----------|---------|---------------------|
| 每侧支撑电容 C_top = C_bot | 2200 μF | 1000~5000 μF | 每个 VSC 子系统内 |
| 变流器直流侧限流电感 L_rect | 1 mH | 0.5~2 mH | 每个 VSC DC 侧串联 |
| 母线电感 L_bus | 2 mH | 1~5 mH | 每条母线馈电支路 |
| 母线电阻 R_bus | 0.05 Ω | 0.01~0.1 Ω | 与 L_bus 串联 |
| 线路电阻（各支路） | 0.5 mΩ | 0.03~3.2 mΩ | 参考原论文表IV范围 |

### M4：整体 PID 调参（最关键阶段）

> ⚠️ 客户明确警告：**单模块调好的 PID 拼在一起会失效，必须整体调参**。
> 本阶段不能抄各模块独立调参的参数，必须从全系统角度重新验证。

**调参顺序（固定，不可改变）**：

```
Step 1 ── 电流内环（全部锁定后再动外环）
   - 对所有 VSC（整流+逆变）分别验证内环阶跃响应
   - 目标：超调 <10%，调节时间 <1ms
   - 若振荡：降低 Kp（步进：乘以 0.7）
   - 若响应慢：增大 Kp（步进：乘以 1.3）

Step 2 ── 整流器电压外环（全发电侧 + 纯阻性负载）
   - 目标：母线 1500V 阶跃响应，超调 <5%，调节时间 <0.2s
   - 若振荡：先降 Kp，再降 Ki
   - 若稳态误差大：增大 Ki

Step 3 ── 逐类负载接入后检验（同 M3 顺序）
   - 每接入一类负载，观察母线扰动恢复时间
   - 若某类负载引起新振荡：先检查该负载逆变器内环，再微调整流器外环
   - 调参原则：不改已通过的内环参数，只调外环 Ki

Step 4 ── 工况切换扰动测试
   - 测试最大功率变化：WC2（巡航高速）→ WC4（DP恶劣天气）
   - 目标：母线电压恢复时间 <0.3s，不出现持续振荡
   - 若不达标：增大整流器电压环带宽（增大 Kp）
```

**M4 证据**：
1. 最终 PID 参数表（截图 init_params.m 关键参数）
2. 四种工况母线电压波形（全部在同一图上对比）
3. WC2→WC4 切换瞬态波形

## 十二、关键风险与规避

| 风险 | 概率 | 规避策略 |
|------|------|---------|
| 整体调参失败导致母线持续振荡 | 高 | 严格按 M4 四步调参顺序，不跳步；出现振荡时先降增益而不是增加带宽 |
| C1 ≠ C2 导致中点漂移 | 中 | init_params.m 统一管理电容参数，C_top 和 C_bot 赋同一变量 |
| 步长过大导致仿真不收敛 | 中 | 首选 2e-6 s；若不收敛改为 1e-6 s；禁用可变步长求解器 |
| 故障仿真后系统不恢复（仿真发散）| 中 | 故障持续时间设置为 50ms，故障后恢复正常运行；不要求故障后自动保护跳闸 |
| MATLAB 2020b 某些 Simscape 模块版本差异 | 低 | 使用基础 Simscape Electrical 模块，避免 R2021b+ 新增 API |
